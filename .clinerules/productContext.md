# Product Context - Mestria Digital

## Por Que Este Projeto Existe

A indústria de extrusão de alimentos é um setor técnico especializado que requer formação prática e teórica actualizada. Existe uma lacuna no acesso a formação de qualidade nesta área específica, especialmente em português.

## Problemas que Resolve

### 1. Formação Técnica Acessível
- Disponibiliza formação técnica especializada em extrusão de alimentos
- Conteúdo em português com contexto industrial real
- Acesso a simuladores práticos

### 2. Gestão de Progresso
- Tracking do progresso individual de cada aluno
- Sincronização entre dispositivos (local + cloud)
- Histórico de conclusão de módulos

### 3. Certificação
- Geração automática de certificados de conclusão
- Verificação de autenticidade via ID único
- Impressão em formato PDF

### 4. Criação de Conteúdo por IA
- Assistente Gemini para geração de cursos
- Processamento de conteúdo manual
- Estruturação automática de currículos

## Como Deve Funcionar

### Fluxo do Utilizador
1. **Registo/Login** → Autenticação via Supabase
2. **Dashboard** → Visualizar cursos disponíveis
3. **Course Detail** → Aceder a lições e módulos
4. **Módulos Interativos** → Prática em simuladores
5. **Certificação** → Receber certificado ao completar

### Experiência do Utilizador
- Interface responsiva (mobile-friendly)
- Design moderno com Tailwind CSS
- Feedback visual de progresso
- Navegação intuitiva

## Objetivos de Experiência do Utilizador

- **Simplicidade**: Interface limpa e focada na aprendizagem
- **Progresso Visível**: Barras de progresso e estatísticas
- **Interatividade**: Módulos práticos com simuladores
- **Acessibilidade**: Funcional em dispositivos móveis
- **Engajamento**: Sistema de ratings e certificados

## Diferenciadores

1. **Foco Nicho**: Único curso de extrusão com simuladores
2. **IA Integrada**: Geração automática de conteúdo
3. **Cloud Sync**: Progresso sincronizado
4. **Certificação**: Emissão automática de certificados

# Project Brief - Mestria Digital E-Learning

## Overview

| Campo | Descrição |
|-------|-----------|
| **Nome do Projeto** | Mestria Digital |
| **Tipo** | WebApp (Single Page Application) |
| **Stack** | React 19 + TypeScript + Vite |
| **URL Repositório** | https://github.com/simaopteixeira-wq/elearning |

## Objetivos do Projeto

Plataforma de e-learning para formação técnica especializada na indústria de extrusão de alimentos, permitindo:
- Gestão de cursos e progresso de alunos
- Módulos interativos práticos (simuladores)
- Geração assistida de cursos por IA (Gemini)
- Certificação de conclusão

## Funcionalidades Principais

1. **Autenticação** - Login/registo via Supabase Auth
2. **Dashboard** - Listagem e pesquisa de cursos
3. **Course Detail** - Player de vídeo/aulas com progresso
4. **Módulos Interativos** - Simulador de extrusão (iframe)
5. **Certificados** - Geração e impressão de certificados
6. **Painel Admin** - Gestão de cursos e sincronização GitHub
7. **IA Course Generator** - Criação automática de cursos via Gemini

## público-Alvo

- Profissionais da indústria alimentar
- Estudantes de tecnologia alimentar
- Formadores em extrusão industrial
